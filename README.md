## How To Use

### Installation

Install all npm dependencies

`npm install`

Install nodemon globally

`npm install -g nodemon`



## Features
#### CRUD (Create, Read, Update and Delete)
- Authentication with Firebase (Reset Password with email)
   - Register
   - Login
   - Forgot Password
   - Login with Google

- Pagination and search where necessary

- Filter Products
   - Filter By Price
   - Filter By Text
   - Filter by Ratings
   - Filter by Category
   - Filter by Sub-Category
   - Filter by Shipping Method
   - Filter by Brands
   - Filter by Color
   
- Products
   - Create Products
   - Fetch Products
   - Rate Product
   - Buy Products
   - Update Products
   - Delete Products

- Payment
   - Pay With Cards
   - Pay on Delivery

- Coupons
    - Create Coupon Codes
    - Apply Coupon Codes on Products

- Orders
    - Place an Order
    - Get Previous Orders
    - Print Order Information

- Delivery
    - Choose Delivery Method
    - Check Delivery Status

- Settings
    - Modify Profile 
    - Change Password



#### API Reference
Extensive and testing documentation with postman

Rename the config/.env.example .env file both on frontend and backend. Modify your environment variables, mongodb uri, set your firebase keys.

PORT

DATABASE
